# kraft-paper-iterm2-profile
    
        
这是我平时用的一个主题，知乎上有朋友要，我就分享出来了，图是网上的，侵权请告诉我我会删除。

文件是 iTerm2 的配置文件，直接导入即可，记得把背景设置成我提供的。

zsh 我推荐使用 oh-my-zsh 并使用 agnoster 主题，效果是不错的。
